Privacy notice

Your recording and transcript are sent to OpenAI to create your focus plan. The temporary audio file is deleted from our server after processing. Please do not share confidential, highly sensitive, or personally identifying information.

How your data is processed
  • Your recording is uploaded securely to our server and sent to OpenAI for speech-to-text processing.
  • The temporary audio file is deleted from our server after processing, whether the request succeeds or fails.
  • The transcript is sent to an OpenAI language model to identify your task, session goal, focus duration, and suggested first step.
  • This prototype does not currently create a user account or permanently store your recording, transcript, or focus plan in its own database.
  • Data sent through the OpenAI API is not used to train OpenAI models unless the API account owner has explicitly opted in to data sharing.
  • OpenAI may temporarily retain some API data for service operation and abuse monitoring, depending on the API endpoint and account configuration.
  • Please avoid mentioning passwords, financial information, health information, confidential work content, or other highly sensitive personal data.

Voice recording requires consent to the processing described above.
